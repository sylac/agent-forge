# sdd-research

Research project exploring Specification-Driven Development practices, agent harness designs, and the synthesis of declarative product documentation with modern agent orchestration.

## Documentation Discipline

Feature specifications live under `Docs/Specs/[Group]/[FeatureId]-[Name].spec.md`. Specs describe the target state of the system in declarative voice. They are updated in place when behavior changes - never converted into change-logs or task lists. Delegate spec work to the `spec-architect` skill.
