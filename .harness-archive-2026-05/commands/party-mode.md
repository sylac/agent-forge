# Party Mode

Facilitates structured, multi-perspective ideation for a single topic.

## Workflow

1. Load skill: `party-mode`
2. Require brainstorming topic as input.
3. Run multi-persona ideation rounds.
4. Consolidate convergent and divergent ideas.
5. Return prioritized recommendations.

## Dispatch Model

- Explicit command dispatch only.
- Never auto-trigger from message detection.
